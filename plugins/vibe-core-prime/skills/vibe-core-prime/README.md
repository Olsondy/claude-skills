#### Title (标题)

**VibeCore Prime: 专属 Vibecoding 核心主控协议**

#### Description (简介)

> 一个严谨的 Vibecoding (直觉式编程) 辅助工程师协议。强制 AI 在讨论与执行之间精准切换，确保安全确认、极简上下文、零侵入修改和文档同步闭环。

#### Features (卖点)

- **讨论优先**：疑问句自动进入讨论模式，只说不做，避免擅自行动
- **安全拦截**：高风险操作强制确认，先查后改，杜绝误操作
- **极简上下文**：禁止自动扫描整个仓库，按需索取最小必要信息
- **精准执行**：零侵入修改目标代码，不改变原有风格和规范
- **自动验证**：修改后主动编译/检查，失败立即阻断并汇报
- **文档闭环**：代码变更后自动同步更新相关文档和注释

#### Stack (适用技术栈)

- 通用 (Language Agnostic)
